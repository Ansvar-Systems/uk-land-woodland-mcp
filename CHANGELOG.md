# Changelog

## 0.1.0 (2026-04-03)

Initial release.

- 10 tools (3 meta + 7 domain)
- Hedgerow regulations: 8 rules covering removal, trimming, laying, coppicing, penalties
- Felling licence rules: 10 scenarios covering thresholds, exemptions, application process
- SSSI operations: 9 operation types with consent process and penalties
- Rights of way: 7 path type entries with minimum widths and reinstatement deadlines
- Common land: 5 activity entries with consent requirements
- Planting guidance: 6 entries covering EWCO grants, EIA screening, ancient woodland buffers
- FTS5 full-text search across 20 indexed entries
- Streamable HTTP + stdio transport
- Docker support with health check
